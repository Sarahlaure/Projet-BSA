# Projet-BSA
Projet réalisé dans le cadre de l’hackathon du Bureau des Statistiques de l’ENSAE Dakar. Il s’agit d’une application R Shiny interactive permettant de visualiser les conflits dans l’UEMOA à partir des données géoréférencées du UCDP, combinant stats exploratoires, cartographie et analyse multivariée.
